1. '이것을 이용!' 씬을 복사하여 하나 추가해서 추가한 씬에다가 스테이지를 구성하십시오!
   절대로 '이것을 이용!'씬에다가 직접 구성하기 금지! 
   추가한 씬은 '컨펌 전' 폴더 안에 넣어주세요! 씬 이름은 편하게 해도 됩니다. 

2. 타일은 Hierachy의 grounds 오브젝트 안에 있는 Ground_Forest 오브젝트를 복붙하여 사용하십시오.
   여기서 T를 누른 상태로 오브젝트를 클릭해서 잡아 끌면 움직여지는데 
   v를 누른 채로 잡아끌면 다른 오브젝트의 모서리로 이동하게 됩니다. 
   하나를 모서리로 옮기고 다른 하나를 추가로 복붙해서 다시 모서리로 옮기는 방식으로 하면 
   예쁘게 정렬 할 수 있습니다. 
   그리고 타일들은 모두 grounds 오브젝트 안에 넣어주세요! 

3. 에셋 폴더 Prefabs 폴더에 있는 프리팹들을 끌어다가 배치하면
   프리팹에 연결된 채로 오브젝트가 맵에 배치가 됩니다. 
   절대 이 연결이 끊어지지 않게 오브젝트를 배치해주십시오.
   Prefabs 폴더 안에서 enemies과 objects 폴더만 사용하면 됩니다!
   오브젝트들은 모두 objects 오브젝트 안에 넣어주세요! 

4. 그 외 주의 사항들 

	아직 피격이 없을 겁니다. 
	플레이어가 오브젝트를 통과하여 지나가도록 해놨습니다. 
	열쇠는 그냥 배치만 해놓으세요! 
	포탈도 작동 안 할건데 배치만 해놓으세요! 
	타일과 오브젝트만 하면 됩니다. 배경이나 envirments 구성은 제 친구가 해줄 겁니다. 

5. 플레이버튼을 누르면 바로 테스트 가능하도록 해놓았습니다. 이걸로 테스트 하세요!

   ***스테이지 구성***

   각 스테이지에는 스테이지의 이유가 있습니다. 
   이유가 있는 스테이지는 그 이유에 맞춰 구성해야 합니다. 




   사용 가능 적: 일반 신라병정(근접), 일반 신라병정(원거리), 황금 신라병정(근접), 황금 신라병정(원거리), 불의 정령(몹)
   사용 가능 오브젝트: 빈 하트, 하트 반쪽, 하트 한쪽, 불 정령, 물 정령, 바람 정령, 바위 정령, 나무 장애물, 불 장애물, 열쇠, 포탈, 움직이는 플랫폼, 


   스테이지 구성 예상도
   1. 튜토리얼 페이지 (내가 함)   
   이번에 해금: 포탈, 빈 하트, 하트 한쪽, 
   사용 가능 적: 없음.
   사용 가능 오브젝트: 포탈, 빈 하트, 하트 한쪽
   
   2. 불 정령을 처음 사용해 보는 스테이지. 시작하자마자 좀만 걸어가면 불 정령과  움직이는 플랫폼과 나무 장애물들을 겪으며 깨는 스테이지
   이번에 해금: 불 정령, 나무 장애물, 움직이는 플랫폼
   사용 가능 적: 없음.
   사용 가능 오브젝트: 포탈, 빈 하트, 하트 한쪽, 나무 장애물, 움직이는 플랫폼
   
   3. 처음 신라병정(근접)이 등장하는 스테이지, 살짝 구석으로 가면 숨겨진 빈 하트가 있음, 포탈로 가는 길에 하트 한쪽 있음
   이번에 해금: 일반 신라병정(근접), 
   사용 가능 적: 일반 신라병정(근접)
   사용 가능 오브젝트: 포탈, 빈 하트, 하트 한쪽, 나무 장애물, 움직이는 플랫폼
   
   4. 점프해서 넘어가서 착지할 만한 자리에 신라병정(근접)이 있음. 여우에게서 도망간 죄로 없애고 지나가야 함. 
	  신라병정(근접)이 여러 마리 등장해도 괜찮은 스테이지, 이 스테이지는 무조건 다른 데로 빠지는 샛길 없이 직선으로 구성
   이번에 해금: 없음.
   사용 가능 적: 일반 신라병정(근접)
   사용 가능 오브젝트: 포탈, 빈 하트, 하트 한쪽, 나무 장애물, 움직이는 플랫폼
   
   5. 오른쪽으로 가다보면 포탈 앞을 불의 정령(몹)이 지키고 있음. 맵의 다른 쪽을 탐험하다 보면 물 정령 발견 가능. 보이는 곳 하트 한 쪽 있음. 그리고 구석에 빈 하트 하나 숨겨놔도 좋을 듯. 
   이번에 해금: 불의 정령(몹), 물 정령
   사용 가능 적: 일반 신라병정(근접), 불의 정령(몹)
   사용 가능 오브젝트: 포탈, 빈 하트, 하트 한쪽, 나무 장애물, 움직이는 플랫폼
   
   6. 신라병정(원거리)가 처음 등장. 점프해서 넘어가서 착지할만한 자리에 이번에는 신라병정(원거리)가 있음. 물의 구체로 배리어를 치고, 
	  여우에게서 도망간 죄로 없애고 지나갈 수 있도록 하기. 신라병정 근접과 원거리가 혼합해서 나오고, 열쇠 하나가 처음 나옴. 이번 스테이지 불의 정령(몹) 사용 금지, 하트 사용 금지.
   이번에 해금: 일반 신라병정(원거리), 열쇠
   사용 가능 적: 일반 신라병정(근접), 일반 신라병정(원거리)
   사용 가능 오브젝트: 포탈, 열쇠, 빈 하트, 하트 한쪽, 나무 장애물, 움직이는 플랫폼
   
   7. 불 장애물 처음 등장, 불의 정령(몹)도 최소 두 개 이상 등장. 불과 관련된 테마라는 느낌의 스테이지 신라병정 근접과 원거리도 조금 나옴. 열쇠는 두 개. 
   이번에 해금: 불 장애물

   사용 가능 오브젝트: 포탈, 열쇠, 빈 하트, 하트 한쪽, 나무 장애물, 불 장애물, 움직이는 플랫폼
